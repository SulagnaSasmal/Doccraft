# DocCraft Project Manifesto

## Context
Evolving from a PDF utility into an **Agentic Document OS**. 
User: Senior Documentation Architect (20y exp) in FinTech/Enterprise.

## Design Identity: "The Business Hub"
- **Vibe:** Technical, Trustworthy, Integrated.
- **Colors:** Slate-950 (BG), Blue-600 (Primary), Slate-800 (Cards).
- **Layout:** Three-column (Nav | Workspace | Insights).

## Functional Shift
- **Yesterday:** A website with PDF buttons.
- **Today:** A central hub where PDFs are "ingested" into structured help-center articles.
- **FinTech Layer:** Every doc must track compliance/PII status.
